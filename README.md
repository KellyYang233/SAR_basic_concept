# SAR_basic_concept
Some concepts about SAR image and image formation
## SAR成像原理
SAR利用脉冲压缩技术获得高的距离向分辨率，利用合成孔径原理获得高的方位向分辨率，从而获得大面积的高分辨率雷达图像。
